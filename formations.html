🔧 Développé

Assistant multi-agences

Backend API (FastPython) + Application mobile Android / iOS

Application d'assistant personnel pour agents immobiliers multi-agences, pilotée par IA (Groq). 12 modules spécialisés :

  • Marché — prix DVF, tendances, estimations (ML RandomForest)
  • CRM — contacts, relances automatisées, matching acheteurs↔mandats
  • Calendrier — synchronisation Google Calendar, visites, rendez-vous
  • DPE — données ADEME, tournées GPX, scoring par quartier
  • Mandats — CRUD, vérification prix vs marché, photos, QR code
  • Négociation — offres, contre-offres, timeline vente
  • Visites — enregistrement, feedback, statistiques
  • Prospection — ciblage DPE F/G × DVF, scoring automatique
  • Financement — calcul mensualité, capacité d'emprunt
  • Rapports — synthèse hebdomadaire, export PDF
  • Documents — gestion documentaire, génération automatique
  • Notifications — push Firebase, alertes intelligentes
